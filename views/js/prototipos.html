Conhecendo o Prototype

  • Conceito de herança
  • Carregado em memória (Performance)
  • Pouco legível
function Pessoa(nome, telefone) { this.nome = nome; this.telefone = telefone; } var gabs = new Pessoa('Gabriel', '8599999-9999'); console.log(gabs instanceof Pessoa ? 'É Pessoa' : 'Não é pessoa'); Pessoa.prototype = { constructor: Pessoa, imprimir: function() { console.log(this.nome+" - "+this.telefone); } }

Má prática

É uma má prática sobrescrever os prototipos do javascript, o ideal é criar seus proprios prototypes.
