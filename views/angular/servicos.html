Serviços

  • Lógica de persistencia
  • Camada entre os Controllers e os back-ends
  • Trabalham com promises
  • Instanciados como Lazy – Angular só instancia quando alguém depende do serviço
  • Singletons – Só existirá uma instância de um serviço para toda a aplicação e é criada pelo serviceFactory
(function() {
    'use strict'

    var app = angular.module('app');

    app.service('MenuService', ['$http', MenuService]);

    function MenuService($http) {
        var vm = this;

        vm.buscarMenus = function() {
            return $http.get('json/menus.json')
        }

        return vm;
    }

})();
